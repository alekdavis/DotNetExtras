      SafeException Base exception class that can be used to filter exceptions for the purpose of returning error information to the caller. If you derive an exception from this class, then calling the GetMessagesT(Exception, Boolean) extension method passing SafeException as the generic type, will only return messages from the exceptions derived from this class. This can help you control the error details sent to the other apps, and make sure no sensitive info is leaked outside of your app.
