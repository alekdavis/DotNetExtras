## Remarks
If the name does not include any compound parts, it will be returned as-is.

## Example


**C#**  
``` C#
// PRINT: Parent.ChildProp

// PRINT: parent.childProp

// PRINT: Parent.ChildProp

// PRINT: parent.childProp
```
