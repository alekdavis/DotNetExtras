## Example


**C#**  
``` C#
List<int> numbers = new List<int>(){ 1, 2, 3, 4, 5 };

// Output: 1, 2, 3, 4, 5
Console.WriteLine(numbers.ToCsv()); 

List<string> words = new List<string>(){ "apple", "banana", "cherry" };
// Output: "apple", "banana", "cherry"
Console.WriteLine(words.ToCsv(", ", "\"", "\""));
```
